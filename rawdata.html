Coordinate Files

Each ZIP archive listed below represents a nonredundant ProPairs data set. The archives contain PDB coordinate files of the binding partners in bound (suffixes "b1.pdb" and "b2.pdb") and unbound state (suffixes "u1.pdb" and "u2.pdb"). In case only one unbound structure is available the archive contains an empty file with the suffix "u2.pdb_not_available".

Data Set: 03/2015

  • small ProPairs set (2,409 interfaces)
    redundancy removed; contains only representative complexes and unbound structures

Data Set: 11/2013

  • small ProPairs set (2,070 interfaces)
    redundancy removed; contains only representative complexes and unbound structures

Flat-File Tables

Each file listed below represents a ProPairs data set. The files are human-readable flat-file tables with each row containing information about one detected interface and one corresponding unbound structure.
Here you can find a description of the table columns.

Data Set: 03/2015

  • large ProPairs set (13,558 interfaces)
    no redundancy removed - but contains clustering information
  • small ProPairs set (2,409 interfaces)
    redundancy removed; contains only representative complexes and unbound structures

Data Set: 11/2013

  • large ProPairs set (11,600 interfaces)
    no redundancy removed - but contains clustering information
  • small ProPairs set (2,070 interfaces)
    redundancy removed; contains only representative complexes and unbound structures
