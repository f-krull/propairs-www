Coordinate files (*.pdb)

The tar.gz archives listed below contain representative, non-redundant interfaces that were identified by ProPairs.

Each interface is represented by four coordinate files (PDB format); both binding partners in bound state (suffixes "b1.pdb" and "b2.pdb") and in unbound state (suffixes "u1.pdb" and "u2.pdb"). In case only one unbound structure was identified by ProPairs, the archive contains an empty file with the suffix "u2.pdb.not_available".

If you are interested in more structures, please refer to the comprehensive data set.
